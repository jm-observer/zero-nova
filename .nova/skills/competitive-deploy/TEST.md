# Competitive Deploy Skill 测试脚本

## 测试概述

本脚本用于验证 `competitive-deploy` skill 的核心功能。

### 测试清单

| 测试 ID | 场景 | 验证内容 |
|--------|------|---------|
| T1 | 基础部署 | 正确识别 3 个竞品 |
| T2 | 命令检查 | 关键命令缺失时跳过竞品 |
| T3 | 混合部署 | 支持 Docker + 源码混合 |

### 执行步骤

1. 读取 SKILL.md 并验证格式
2. 模拟用户输入部署方案
3. 执行命令检查逻辑
4. 验证输出格式
