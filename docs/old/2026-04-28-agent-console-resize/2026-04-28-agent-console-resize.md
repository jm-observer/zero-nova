# 2026-04-28 Agent Console Resize

## 时间
- 创建：2026-04-28
- 最后更新：2026-04-28

## 项目现状
- `deskapp` 中的 `agent-console` 当前固定为右侧 `360px` 面板。
- 桌面端布局通过 `#workspace.console-open { margin-right: 360px; }` 为控制台预留空间。
- 左侧 `sidebar` 已支持拖拽调整宽度，但 `agent-console` 尚未支持同类交互。

## 整体目标
- 为桌面端 `agent-console` 增加左右拖拽能力。
- 拖拽后控制台宽度与 `workspace` 右侧预留空间保持同步。
- 宽度写入本地存储，刷新后恢复。
- 移动端保持现有覆盖式布局，不启用拖拽。

## Plan 拆分
- Plan 1：控制台拖拽交互与样式适配
  - 内容：增加拖拽手柄、宽度状态恢复、桌面端宽度同步、边界限制。
  - 依赖：无。
  - 顺序：1。

## 风险与待定项
- 若 `workspace` 右侧预留宽度与控制台宽度不同步，会导致聊天区布局错位。
- 需要保证移动端媒体查询优先级高于桌面端宽度变量。
